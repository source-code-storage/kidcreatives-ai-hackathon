## Tips

1. **Be specific**: "Show me Supabase storage upload syntax" is better than "How does Supabase work?"
2. **Ask for examples**: "Show me an example of..." gets code snippets
3. **Reference your stack**: Mention React, TypeScript, etc. for relevant answers
4. **Latest APIs**: context7 fetches current documentation automatically
